Bachelor's Final Thesis

Context: This project is driven by a passion for plants and gardening and a desire to find a solution to struggles with keeping plants alive. I started having an Arduino in my house and me not knowing what I could do for my bachelor's final thesis, so I came up with this project with the goal is to improve the success rate of gardening by accurately monitoring and controlling growing conditions, making it easier for people to grow and care for their plants.
Project's Description: Design and implementation of a personal climate control system for plants using cloud computing and IoT technologies, with the aim of accurately monitoring and controlling the growing conditions of plants in a personal greenhouse. View Project

Tasks:
• Conducted research on the use of cloud computing and IoT technologies for personal gardening.
• Designed the architecture of the personal climate control system.
• Programmed the microcontroller (Arduino) to collect data from various sensors and communicate with the cloud platform.
• Conducted experiments to test the system's performance and accuracy.
• Conducted a literature review on the use of technology in agriculture.
• Selected appropriate sensors for monitoring temperature, humidity, soil moisture, and light intensity.
• Programmed the microcontroller to control fans, heaters, and lights based on the collected data.
• Developed an application using Python for the user being able to use the data.
• Wrote a report documenting the design, implementation, and evaluation of the system.

Results: The personal climate control system successfully monitored and controlled the growing conditions of plants, resulting in improved yields and increased efficiency. The project demonstrated the benefits of using cloud computing and IoT technology for personal gardening, such as cost savings and automation. The project also highlighted the importance of open-source software and low-cost hardware components for cost-effective implementation.
• The system was able to accurately monitor and control the growing conditions of plants, resulting in improved yields and increased efficiency.
• Successfully achieved all goals in designing and implementing the system.
• IoT and cloud computing technologies were employed to enable monitoring and a response mechanism.
• Cost-effective, easy-to-use, and highly functional system created.
